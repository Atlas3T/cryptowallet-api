# cryptowallet-api

## License

  GPLv2
  
## Install the dependencies
```bash
  npm install
```
### Add .env file
See [.env.example](https://github.com/atlascityk/cryptowallet-api/blob/develop/.env.example)

### Build the app for production
```bash
  npm run build
```

### Start the API
```bash
  npm run start
```


## Docker
```bash
  docker-compose build

  docker-compose up
```
